How to install on your Vita

The bundler produces superhexagon.zip. Inside it are the loader VPK and a superhexagon/ data folder extracted from your APK. Super Hexagon can be bought from Terry Cavanagh on itch.io.

  1. Unzip superhexagon.zip on your computer.
  2. Upload the superhexagon/ folder to ux0:data/ so the Vita has ux0:data/superhexagon/assets/ and the four required Android libraries beside it.
  3. Copy superhexagon_vita.vpk to the Vita and install it with VitaShell.
  4. Launch Super Hexagon Vita from the LiveArea.

Bundler runs entirely in-browser via fflate.
